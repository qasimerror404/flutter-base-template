## ğŸ”§ Key Concepts

### Clean Architecture Layers

#### 1. **Domain Layer** (Business Logic)
- **Entities**: Pure Dart classes representing business objects
- **Repository Interfaces**: Contracts for data operations
- **Use Cases**: Business logic (optional for simple operations)

#### 2. **Data Layer** (Data Management)
- **Models**: Data transfer objects with JSON serialization
- **Data Sources**: Remote (API) and Local (Isar) data sources
- **Repositories**: Implementation of domain repository interfaces

#### 3. **Presentation Layer** (UI)
- **Providers**: Riverpod state management
- **Pages**: Screen widgets
- **Widgets**: UI components

### State Management with Riverpod

This template uses **manual AsyncNotifier** (no code generation):

```dart
final productsProvider = AsyncNotifierProvider<ProductsNotifier, List<Product>>(
  () => ProductsNotifier(),
);

class ProductsNotifier extends AsyncNotifier<List<Product>> {
  @override
  Future<List<Product>> build() async {
    // Initial load
    return await _fetchProducts();
  }

  Future<void> refresh() async {
    state = const AsyncValue.loading();
    state = await AsyncValue.guard(() => _fetchProducts());
  }
}
```

### Error Handling

Using **Either** pattern from `dartz`:

```dart
Future<Either<Failure, Product>> getProduct(String id) async {
  try {
    final product = await api.getProduct(id);
    return Right(product);  // Success
  } catch (e) {
    return Left(ServerFailure(e.toString()));  // Failure
  }
}
```

### Offline-First with Isar

Data flow:
1. Try to fetch from API
2. Save to local Isar database
3. On failure, return cached data
4. Sync when back online

1. **Create feature folder structure:**
   ```bash
   mkdir -p lib/features/my_feature/{data/{datasources,models,repositories},domain/{entities,repositories},presentation/{providers,pages,widgets}}
   ```

2. **Create entity** (domain/entities/):
   ```dart
   class MyEntity extends Equatable {
     final String id;
     final String name;
     
     const MyEntity({required this.id, required this.name});
     
     @override
     List<Object?> get props => [id, name];
   }
   ```

3. **Create model** (data/models/):
   ```dart
   @collection
   class MyModel {
     Id id = Isar.autoIncrement;
     late String name;
     
     MyEntity toEntity() => MyEntity(id: id.toString(), name: name);
     factory MyModel.fromEntity(MyEntity entity) => MyModel()..name = entity.name;
   }
   ```

4. **Create repository interface** (domain/repositories/)
5. **Create data sources** (data/datasources/)
6. **Create repository implementation** (data/repositories/)
7. **Create providers** (presentation/providers/)
8. **Create pages & widgets** (presentation/)
9. **Add routes** to `lib/core/router/app_router.dart`
