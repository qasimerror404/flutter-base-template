# Screenshots

This folder contains screenshots of the Flutter Core Template application.

## Adding Screenshots

To showcase your app on GitHub:

1. Take screenshots of key features
2. Save them in this folder with descriptive names:
   - `light.png` - Light theme example
   - `dark.png` - Dark theme example
   - `rtl.png` - RTL (Arabic) layout
   - `products.png` - Products list page
   - `detail.png` - Product detail page
   - `search.png` - Search functionality
   - `favorites.png` - Favorites page
   - `settings.png` - Settings page

3. Recommended dimensions: 1080x2400 (mobile) or 1920x1080 (web)

4. Update the main README.md to reference your screenshots

## Screenshot Guidelines

- Use real, production-quality content
- Ensure text is readable
- Show key features and UI elements
- Maintain consistent style across screenshots
- Include both light and dark themes
- Show responsive design (mobile/tablet)
- Demonstrate localization (EN/AR)

## Tools for Screenshots

### Mobile
- Android: Built-in screenshot tool or ADB
- iOS: Simulator → File → New Screen Shot
- Device: Power + Volume buttons

### Editing
- [Figma](https://figma.com) - Frame screenshots
- [Cleanshot](https://cleanshot.com) - Mac screenshot tool
- [Flameshot](https://flameshot.org) - Linux/Windows
- [Shotsnapp](https://shotsnapp.com) - Add device frames online

### Device Frames
- [MockUPhone](https://mockuphone.com)
- [Screely](https://www.screely.com)
- [Device Frames](https://deviceframes.com)
